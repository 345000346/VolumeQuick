# VolumeQuick - 快捷音量控制工具

一个简单而高效的 Windows 音量控制工具，让您通过鼠标在屏幕左上角快速调节系统音量。

## ✨ 特性

- 🎮 在屏幕左上角快速调节系统音量
- 🖱️ 使用鼠标滚轮直观控制
- 🔇 中键快速静音/取消静音
- ⚙️ 可自定义音量调节步进值
- 🚀 支持开机自启动
- 💡 低资源占用，运行流畅

## 🚀 使用方法

1. 将鼠标移动到屏幕左上角
2. 使用鼠标滚轮上下滚动调节音量
3. 点击鼠标中键切换静音状态
4. 按下 `Ctrl + Alt + O` 打开设置界面

## ⚙️ 设置选项

- 音量调节步进值：调整每次滚动改变的音量大小（2-20%）
- 开机自启动：设置是否随系统启动

## 💻 系统要求

- Windows 10/11
- [AutoHotkey v2.0](https://www.autohotkey.com/) 或更高版本
- 不需要管理员权限

## 📥 下载和安装

1. 下载最新版本的发布包
2. 解压到任意位置
3. 运行 `VolumeHotkey.ahk`
4. 首次运行时可选择是否开机启动
